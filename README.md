# Alisha-sports-Academy
A simple and responsive Sports Academy website built using HTML and CSS. This project showcases a modern landing page for a sports academy with information about training programs, facilities, and contact details.

## 🚀 Features

- Responsive design
- Clean and modern UI
- Sports academy landing page
- Easy navigation
- Beginner-friendly HTML & CSS project

## 🛠️ Technologies Used

- HTML5
- CSS3
